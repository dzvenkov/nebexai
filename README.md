# NebexAI Summarizer

A simple API service that takes a GitHub repository URL and returns a human-readable summary generated by an LLM.

## Features
- FastAPI-powered asynchronous API.
- Intelligent repository fetching and filtering.
- LLM-powered project summarization (Nebius Token Factory or alternative).

## Requirements
- Python 3.10+

## Setup Instructions

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/dzvenkov/nebexai.git
   cd nebexai
   ```

2. **Create and activate a virtual environment**:
   ```powershell
   # Windows
   python -m venv .venv
   .\.venv\Scripts\activate
   ```
   ```bash
   # Linux/macOS
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Variables**:
   Create a `.env` file in the root directory (optional, or export directly):
   ```bash
   NEBIUS_API_KEY=your_api_key_here
   ```

## Running the Server

Start the server using `uvicorn`:

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

The server will be available at `http://127.0.0.1:8000`.

## Testing the API

You can test the `/summarize` endpoint using `curl`:

```powershell
curl -X POST http://127.0.0.1:8000/summarize `
     -H "Content-Type: application/json" `
     -d '{\"github_url\": \"https://github.com/psf/requests\"}'
```

## Project Structure
- `app/main.py`: Main FastAPI application entry point.
- `app/github.py`: Logic for interacting with GitHub API and filtering repository contents.
- `requirements.txt`: Project dependencies.
- `docs/`: Project documentation and task tracking.
